# Full-Stack Teknologioversigt

| Teknologi | Kategori | Primær anvendelse | Nøglepunkter |
|-----------|----------|-----------------|---------------|
| Angular | Frontend framework | Store webapps | Typescript, struktureret, opinionated |
| React | Frontend bibliotek | UI-komponenter | Component-driven, fleksibelt, stort økosystem |
| Next.js | React full-stack framework | SSR, SSG, full-stack | File-based routing, API routes, serverless functions |
| NestJS | Backend framework (Node) | Enterprise API’er | Angular-lignende struktur, modulært, DI |
| Node.js | JavaScript runtime | Backend | Event-driven, non-blocking |
| .NET Core Web API | Backend framework | REST API’er | Hurtigt, cross-platform, C# |
| Blazor | Frontend framework (.NET) | Webapps i C# | WebAssembly eller server-rendered |
| C# | Programmeringssprog | Backend, desktop, spil | Strongly typed, .NET integreret |
| JavaScript / TypeScript | Programmeringssprog | Browser / backend / full-stack | Dynamisk / strongly typed |
| HTML / CSS / Sass / SCSS | Styling / markup | Struktur og styling af websider | Grundlæggende styling, variabler, nesting |
| Tailwind CSS / Bootstrap / Material UI / MudBlazor / Ant Design / Chakra UI | Styling / UI bibliotek | Styling og komponenter | Prebuilt komponenter, utility-classes, konsistent design |
| Docker / Kubernetes | DevOps | Containerization & deployment | Isolation, scaling, reproducible builds |
| Apache / Nginx | Webserver / reverse proxy | Hosting, load balancing | Modulbaseret, hurtig, lavt memory-forbrug |
| MVC (.NET MVC) | Arkitektur / framework | Webapps med Views & Controllers | Separation of concerns |
| MS SQL / PostgreSQL / MySQL / SQLite / MongoDB / Redis | Database | Data storage | Relationel, NoSQL, key-value, embedded, caching |
| Prisma | ORM / Database toolkit | Database abstraction | Works with Postgres, MySQL, SQLite, type-safe |
| In-memory DB (f.eks. SQLite in-memory, Redis) | Test / Caching | Hurtig midlertidig database | Ideel til unit testing og prototyping |
| Postman / Swagger / Scalar / GraphQL / Apollo | API / dokumentation | Test og dokumentation af API’er | REST & GraphQL support, UI til test |
| Firebase / Supabase | Backend-as-a-Service | Auth, DB, Hosting | Realtime database, serverless, open-source |
| Electron | Desktop apps | Cross-platform desktop apps | Node + Chromium |
| Vite / Webpack / Parcel | Build tools / bundler | Pakning af frontend | Hot reload, bundling, optimering |
| Jest / Mocha / Chai / Cypress / Playwright | Test frameworks | Unit, integration, end-to-end testing | Automation, test coverage, browser & server tests |
| JWT (JSON Web Token) | Auth / sikkerhed | Token-baseret authentication | Stateless, signeret token |
| bcrypt / Argon2 | Kryptering / password hashing | Sikker lagring af passwords | Salted hash, industry standard |
| OAuth2 / OpenID Connect | Auth / sikkerhed | Single sign-on / third-party auth | Standard protokoller til autentifikation |
