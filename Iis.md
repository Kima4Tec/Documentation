2️⃣ Opret forbindelse til Simply.com-serveren

I menuen til venstre: højreklik på Start Page → vælg Connect to a Site… (eller “Connect to a Server” afhængigt af version)

Indtast oplysningerne, som Simply.com har givet

#### Login oplysninger er de samme som til FTP og Web Deploy:

Server: [webserver] (Kan også ses i oprettelsesmail samt i vores kontrolpanel)
Site/Application: [domain]
User name: [username]
Password: [password]

Klik OK → IIS forsøger at oprette forbindelse

Hvis forbindelsen lykkes, vil du nu se serveren og dine sites under “Connections”
